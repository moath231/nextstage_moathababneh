## Prerequisites

- PHP 7.4 or higher
- Composer
- MySQL


## Installing

- Clone the repository to your local machine: <br>
  `git clone https://github.com/moath231/nextstage_moathababneh`

- Install dependencies: <br>
 `cd nextstage_moathababneh`<br>
  `composer install`

- compy a .env.example :
  `sudo cp .env.example .env`

- `composer install`

- `php artisan migrate`

- `php artisan db:seed`

- `php artisan serve`
